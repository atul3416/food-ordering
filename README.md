# Food Ordering System

A full-stack food ordering platform built using Django, Django REST Framework, and React.js. The application allows users to browse food items, place orders, manage wishlists, submit reviews, and track order status, while administrators can monitor business performance through a custom analytics dashboard.

## Features

### User Features

* User Registration & Login
* JWT Authentication
* Browse Food Items
* Search and Category-Based Filtering
* Add to Cart
* Wishlist Management
* Star Ratings and Reviews
* Place and Track Orders
* Order Cancellation

### Admin Features

* Custom Admin Dashboard
* Monthly Sales Analytics
* Top-Selling Food Statistics
* User Registration Statistics
* Order Management
* Product Management
* CSV Data Export

## System Workflow

1. Users register and authenticate using JWT tokens.
2. Browse, search, and filter food items.
3. Add products to cart or wishlist.
4. Place orders and track delivery status.
5. Admin manages products, users, and orders.
6. Dashboard provides sales and business insights.
